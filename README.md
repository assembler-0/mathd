# SMCTL - C++ system manager 
__Target distros:__
- Arch-based distro
- _GNU/Linux_
***
__Features list (some might not yet implemented):__
- Calculator
- Calculator - High precision
- Cache cleaner
- TUI
- Temporary files cleaner
- System resources manager
- System autotune
- File marker & finder
- Update system as a service
- Debloat
***
__Usage:__
1. Run `autobuild`
2. Compile from source
